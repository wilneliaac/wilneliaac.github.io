Intro to Programming

➜ Course Overview and Goals

In this course you will learn the foundation of computer science and basic programming, with the goal of helping you develop logical thinking and problem solving skills. In this introductory course we will be moving from block-based programming to text-based programming. You will be exploring some well-known programming languages with an emphasis in Python. Python can be used for many types of projects, ranging from designing websites to examining large amounts of data. In this course, you will learn Python commands, functions, and control structures by solving puzzles and writing creative programs.
