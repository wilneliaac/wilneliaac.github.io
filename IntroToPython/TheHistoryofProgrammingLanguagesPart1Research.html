History of Programming Languages

➜ The History of Programming Languages Part 1 Research Activity

➜ Description of activity

You have been randomly assigned a Programming Language. You will conduct research during 2 class periods about the assigned Programming Language. You will be taking notes of your research using the Google Docs shared by your teacher.

➜ Your research should include

  • Programming Language Name
  • Year created
  • Who created it
  • Primary uses
  • Used by
  • Short description (3-5 sentences) to include
    • What is it? What does it do?
    • History
    • Fun Fact
  • Citations of references
